# Android 16 Bilibili

一个基于 Jetpack Compose 的 Android Bilibili 客户端实验项目，当前已实现首页、关注、直播、历史、搜索、用户主页、视频详情与番剧/合集播放等主要能力。

## 下载

[GitHub Releases](https://github.com/gaoxipeng/Android_16_bilibili/releases/latest) 下载 `app-release.apk` 安装即可。

## 当前功能
- 首页推荐、关注动态、排行榜、历史、收藏
- 视频详情播放、分 P 切换、合集切换、评论浏览
- 番剧搜索与番剧播放
- 直播推荐、关注直播、直播间播放、在线人数刷新
- 直播间弹幕显示、弹幕表情、清屏手势、主播主页跳转
- 用户主页、关注/粉丝列表、资料缓存
- **深色模式**：设置中可选浅色 / 深色 / 跟随系统
- **多账号管理**：设置页切换、添加账号，左滑删除已保存账号
- **弹幕设置**：视频/直播播放器内长按「弹」调节显示区域、不透明度、字号与速度
- 设置页内置使用说明，以及信息流单列/双列切换、后台播放开关

## 运行环境

- Android Studio 最新稳定版
- JDK 11
- Android SDK 37
- 最低系统版本：Android 16

## 本地运行

1. 使用 Android Studio 打开项目根目录 `bilibili`
2. 确认本地已安装 Android SDK 37
3. 如需 release 签名，准备 `keystore.properties`
4. 直接运行 `app` 模块

Debug 编译命令：

```bash
./gradlew :app:assembleDebug
```

## Release 打包

项目已在 `app/build.gradle.kts` 中读取 `keystore.properties` 作为 release 签名配置。

`keystore.properties` 示例：

```properties
storeFile=release-keystore.jks
storePassword=your_store_password
keyAlias=your_key_alias
keyPassword=your_key_password
```

打包命令：

```bash
./gradlew :app:assembleRelease
```

默认输出位置：

```text
app/build/outputs/apk/release/app-release.apk
```

## 使用说明摘要

### 登录与账号

- 本应用通过哔哩哔哩网页登录，首次使用请到「我的 → 设置 → 账号管理」完成登录
- 登录成功、下拉刷新后会在顶部显示胶囊提示
- 设置页可保存多个账号，点击切换当前账号，左滑账号行可删除
- 点击「添加账号」会打开网页登录，完成后保存为新账号

### 浏览与导航

- 底部五个入口：首页、关注、直播、历史、我的
- 首页、关注、历史支持下拉刷新；再次点击当前 Tab 会回到顶部并刷新
- 从首页进入详情或其他 Tab 后返回，五个底部 Tab 均会保留原有滚动位置
- 首页上滑加载更多最多 10 次；下拉刷新后重置
- 向下滚动时底部栏收起到左侧小胶囊，点击可再次展开；上滑可隐藏底部栏，下滑恢复

### 视频播放

- 点击视频卡片进入详情页；信息流内可内联预览
- 从详情页返回后再次播放同一视频，会尽量延续上次进度与画面
- 详情页支持弹幕开关、倍速播放与全屏观看
- 长按控制栏「弹」在播放器内打开弹幕设置（显示区域、不透明度、字号、速度）
- 支持分 P、合集、番剧切换；从简介或评论区进入 UP 主主页时当前视频会自动暂停

### 直播

- 推荐页按分区筛选；关注页展示正在直播的房间
- 点击左上角头像或昵称进入主播个人主页
- 支持飘屏弹幕与底部弹幕列表；长按「弹」可设置弹幕样式
- 全屏时底部为弹幕输入框、弹幕开关与全屏按钮同一行
- 向右滑动清屏，向左滑动或按返回键恢复

### 设置

- 「我的」页右上角齿轮进入设置
- 可切换深色模式（浅色 / 深色 / 跟随系统）
- 可切换信息流单列/双列布局
- 「后台播放声音」关闭后，应用切到后台时会暂停视频
- 设置页「使用说明」可查看更完整的功能介绍
